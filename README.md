# Statistical Inference and Hypothesis Testing in R

This repository contains a comprehensive statistical analysis using R, focusing on probability distributions and hypothesis testing.

## 📊 Analysis Highlights
* **Goodness-of-Fit (Chi-Square):** Evaluated categorical distribution claims using observed vs. expected frequencies.
* **Welch Two-Sample t-test:** Analyzed educational dataset (Baumann) to determine significant differences in teaching method improvements.
* **Confidence Intervals:** Calculated 99% CI bounds for population mean differences.

## 🛠️ Tech Stack
* **Language:** R
* **Libraries:** `carData`
* **Format:** RMarkdown (.Rmd) for reproducible research.
